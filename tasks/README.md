# Tasks

Work items for the mvt-games project. Agents and humans can pick up, track, and
complete tasks here.

## How It Works

- **backlog/** - tasks not yet started. Pick one up by moving it to `active/`.
- **active/** - tasks currently being worked on. Update the progress log as you
  go.
- **archive/** - completed tasks. Move here when all acceptance criteria are
  met.

Task files are numbered (`001-`, `002-`, ...) for easy reference. Each file
contains a description, acceptance criteria checklist, and a progress log.
